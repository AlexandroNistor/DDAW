Listas y diccionarios en Python: desde principiantes hasta expertos

En este documento, exploraremos las listas y diccionarios en Python, comenzando con conceptos básicos y avanzando hacia técnicas más avanzadas. Cubriremos tanto ejemplos prácticos como explicaciones detalladas para ayudarte a dominar estos fundamentales tipos de datos en Python.

Listas en Python

Definición y creación de listas

mi_lista = [1, 2, 3, "hola", True]

Puedes crear listas vacías o con elementos de diferentes tipos.

Acceso a elementos

primero = mi_lista[0]
segundo = mi_lista[1]

Puedes acceder a elementos individuales por su índice. Los índices comienzan desde 0.

Modificación de listas

mi_lista.append("nuevo elemento")
mi_lista[1] = "cambiar valor"

Las listas son mutables, lo que significa que puedes modificarlos después de su creación.

Slicing

parte = mi_lista[1:3]

El slicing te permite obtener subsecuencias de una lista. Este ejemplo obtiene los elementos desde el índice 1 hasta el 3 (excluido).

Métodos de listas

# Ordenar la lista
mi_lista.sort()

# Reversar la lista
mi_lista.reverse()

# Buscar un elemento
si_existe = "hola" in mi_lista

# Contar ocurrencias
contar = mi_lista.count("hola")

# Eliminar un elemento
mi_lista.remove("hola")

Python ofrece varios métodos útiles para trabajar con listas.

Diccionarios en Python

Definición y creación de diccionarios

mi_diccionario = {"nombre": "Juan", "edad": 30, "ciudad": "Madrid"}

Puedes crear diccionarios vacíos o con pares clave-valor de diferentes tipos.

Acceso a valores

nombre = mi_diccionario["nombre"]

Puedes acceder a los valores por su clave. Si la clave no existe, Python levantará una excepción. Para evitar esto, puedes usar el método `.get()`:

nombre = mi_diccionario.get("nombre", "Desconocido")

Modificación de diccionarios

mi_diccionario["edad"] = 31
mi_diccionario["ocupacion"] = "Desarrollador"

# Agregar una nueva pareja clave-valor
mi_diccionario["lenguaje_favorito"] = "Python"

Los diccionarios también son mutables y puedes modificarlos.

Iterar sobre diccionarios

for clave in mi_diccionario:
        print(f"{clave}: {mi_diccionario[clave]}")

for valor in mi_diccionario.values():
    print(valor)

for par in mi_diccionario.items():
    print(par)

Puedes iterar sobre las claves, valores o pares clave-valor de un diccionario.

Avanzando con listas y diccionarios

Listas anidadas

lista_anidada = [[1, 2], [3, 4], [5, 6]]

Puedes crear listas que contengan otras listas. Esta estructura es útil para representar matrices o tablas de datos.

Diccionarios anidados

diccionario_anidado = {"personas": {"juan": 30, "maria": 25}}

También puedes crear diccionarios que contengan otros diccionarios. Esto es útil para organizar datos jerárquicamente.

Listas y diccionarios en funciones

def procesar_datos(datos):
    for dato in datos:
        print(dato)

procesar_datos(mi_lista)

def calcular_imponible(precio):
    impuestos = {
        "IVA": 0.21,
        "IGIC": 0.0405
    }
    total = precio * (1 + impuestos["IVA"])
    return total

precio = 100
total = calcular_imponible(precio)

Puedes utilizar listas y diccionarios como argumentos de funciones.

Mejores prácticas

  1. Usa nombres descritivos para tus listas y diccionarios.
  2. Mantén tus listas y diccionarios organizados y bien estructurados.
  3. Utiliza constantes para valores que se repiten frecuentemente.
  4. Considera usar tipos de datos más específicos cuando sea apropiado (por ejemplo, tuple para secuencias inmutables).
  5. Documenta tu código explicando el propósito y el significado de tus estructuras de datos complejas.
